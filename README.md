## Student Grade Tracker

This assignment focuses on basic Python concepts such as loops, conditionals, and data handling.
It is implemented as a command-line Student Grade Tracker.

## Features

* Cleans raw student data (names, roll numbers, marks)
* Calculates grades, total, and average marks
* Finds highest and lowest scoring subjects
* Allows user to add new subjects with validation
* Generates class performance summary
* Performs string manipulation operations

## Concepts Used

* Loops (for, while)
* Conditional statements (if-elif-else)
* Lists and dictionaries
* String methods
* Basic input validation

## How to Run

1. Open the file in VS Code or any Python IDE
2. Run the file
3. Follow the instructions in terminal
