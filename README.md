## Student Grade Tracker

This assignment demonstrates the use of core Python concepts such as loops, conditionals, lists, and string handling.
The program is designed as a command-line **Student Grade Tracker** that processes student data, performs analysis, and generates reports.

---

## Task Breakdown

### Task 1 — Data Parsing & Profile Cleaning

* Cleans raw student data (names, roll numbers, marks)
* Converts data into proper format
* Validates student names
* Displays formatted student profile cards using f-strings

---

### Task 2 — Marks Analysis

* Assigns grades based on marks
* Calculates total and average marks
* Identifies highest and lowest scoring subjects
* Allows user to add new subjects with input validation

---

### Task 3 — Class Performance Summary

* Calculates average marks for each student
* Determines Pass/Fail status
* Displays formatted class report
* Finds class topper and class average

---

### Task 4 — String Manipulation Utility

* Cleans and formats text
* Counts occurrences of a word
* Replaces words with new text
* Splits text into sentences
* Displays numbered sentences

---

## Concepts Used

* **Loops**: `for`, `while`
* **Conditionals**: `if`, `elif`, `else`
* **Data Structures**: Lists, Dictionaries, Tuples
* **String Operations**: `strip()`, `title()`, `count()`, `replace()`, `split()`
* **Input Validation**: Manual checking of numeric inputs

## How to Run

1. Open the file `part1_grade_tracker.py` in VS Code or any Python IDE
2. Run the program
3. Follow the instructions displayed in the terminal

---

## Output

The program will display:

* Student profile cards
* Subject-wise grades
* Total and average marks
* Highest and lowest scoring subjects
* Updated average after adding subjects
* Class performance summary
* String manipulation results
